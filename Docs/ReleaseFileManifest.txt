Message Dialog Component

Release File Manifest
---------------------

Release 1.0
-----------

PJMessageDialog.pas            : component source file
PJMessageDialog.dcr            : resource file for component palette glyph
PJMessageDialog.hlp            : help file (for Delphi 3 & 4 IDE)
pjmsgdlg.htm                   : information sheet
